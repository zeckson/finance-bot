# Finance bot
Finance bot to handle finance inquiries

# Working with the project
1. To install dependencies use:
```shell
deno add jsr:@luca/cases@1.0.0
```
2. To reload all dependencies locally:
```shell
deno install --reload --lock
```
3. Run application:
```shell
deno task start
```
4. Test application:
```shell
deno task test
```
5. Lint application:
```shell
deno task lint
```
